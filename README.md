Matrix – Fitness Goal Tracker is an interactive front-end web application built using HTML, CSS, and JavaScript. 
The project was developed with the assistance of AI (Claude) for structural guidance and logic suggestions, while implementation, customization, and understanding of the features were completed manually.

The application allows users to:

Set and track daily fitness goals (steps, calories, workout minutes)
Monitor hydration levels with dynamic progress updates
Calculate Body Mass Index (BMI) with instant category feedback
View real-time progress through animated progress bars
The UI follows a futuristic “Matrix”-inspired theme, incorporating neon styling, responsive layouts, and smooth user interactions to create an engaging experience.

This project demonstrates practical knowledge of:

DOM manipulation
Event handling
Form validation
JavaScript calculations and logic building
Dynamic UI updates
It reflects both modern development practices (AI-assisted workflow) and strong understanding of core front-end fundamentals.
